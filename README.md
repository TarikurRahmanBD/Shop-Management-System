# Shop Management System

## Project Overview

This project is a simple desktop-based Shop Management System built with Python and Tkinter. It allows users to add, delete, search, and view items stored in a local database file. The application is designed for a small retail or stationery store and provides a straightforward graphical interface for managing inventory records.

## How to Install & Run

1. Install Python:
   - Download and install Python 3.10+ from https://www.python.org/downloads/.

2. Open a terminal or PowerShell in the project folder:
   ```bash
   cd /d f:\shop-managment-system
   ```

3. Run the application:
   ```bash
   python ShopManagmentSystem_DataFlair.py
   ```

4. The Tkinter GUI window will open. Use the form fields and buttons to manage shop inventory.

> Note: This application uses the built-in Tkinter library and does not require additional external dependencies.

## Features

- **Add Item**: Add a new item record with name, price, quantity, category, and discount.
- **Delete Item**: Delete an item entry from the database by item name.
- **Search Item**: Search for an item by name and display its details.
- **View Database**: Display the current database contents in a scrollable window.
- **Clear Form**: Reset all input fields.
- **Exit**: Close the application safely.
